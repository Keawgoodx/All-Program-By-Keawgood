#คู่มือการใช้งานเบื้องต้น By Keawgood

ข้อควรระวังก่อนเริ่ม: เครื่องคอมพิวเตอร์ต้องติดตั้งโปรแกรม Python ไว้แล้ว (และตอนติดตั้งต้องติ๊กถูกที่ช่อง `Add Python.exe to PATH` ด้วยนะครับ)

เมื่อดับเบิลคลิก `Install.bat` เสร็จแล้ว ให้คลิก `Run.bat` เพื่อเปิดหน้าต่างหลัก (Hub) คุณจะเจอ 4 เมนูย่อย ดังนี้ครับ:

📖 โหลดนิยายอัตโนมัติ (V2)
 หน้าที่: ดูดไฟล์ Text นิยายจากเว็บต้นทาง 
 วิธีใช้: วางลิงก์เว็บนิยาย (หน้าสารบัญ หรือหน้าตอนแรก) -> เลือกโฟลเดอร์ปลายทาง -> กดเริ่มโหลด
 ทริค: หากเว็บกันบอท ให้เลือกโหมด "ไต่ลิงก์ทีละตอน" แล้วกดปุ่ม "ขอ Cloudflare Cookie" ก่อนโหลด

🗂️ จัดการไฟล์นิยาย
 หน้าที่: รวมไฟล์ย่อยเป็นไฟล์เดียว, แยกไฟล์เดียวเป็นตอนย่อยๆ หรือแปลงไฟล์ (TXT, DOCX, PDF)
 วิธีใช้: เลือกโฟลเดอร์ต้นทาง -> เลือกโฟลเดอร์ปลายทาง -> ตั้งชื่อไฟล์ -> กดเริ่ม

✂️ จัดการคำศัพท์นิยาย
 หน้าที่: จัดการไฟล์คลังคำศัพท์ (ทำ Dictionary) โดยโปรแกรมจะช่วยตัดคำที่ซ้ำกันออกให้
 วิธีใช้: ลากไฟล์ `.txt` ที่เก็บคำศัพท์มาวางในช่องซ้าย -> เลือกว่าจะเก็บคำซ้ำแบบไหน (แนะนำ: เก็บอันที่ยาวที่สุด) -> กดปุ่มสายฟ้าเพื่อประมวลผล -> กดเซฟ

🎵 แปลงไฟล์เสียงเป็นวิดีโอ
 หน้าที่: นำไฟล์เสียง (Audio) มาแปะรูปภาพนิ่ง เพื่อทำเป็นวิดีโอ MP4 สำหรับลง YouTube หรือช่องทางอื่นๆ
 วิธีใช้: เลือกไฟล์เสียงทั้งหมด -> เลือกรูปภาพ 1 รูป -> เลือกโฟลเดอร์เซฟ -> ระบุจำนวนคลิปที่จะมัดรวมเป็น 1 ไฟล์ -> กดเริ่มสร้างวิดีโอ
