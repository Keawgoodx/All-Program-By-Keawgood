# Keawgood Universe - All-in-One Hub

โปรแกรมจัดการและดาวน์โหลดนิยายแบบครบวงจร พัฒนาด้วย `Python` และ `CustomTkinter`

## ไฟล์สำคัญในโปรเจกต์

* `Keawgood_Universe.py`
  * ไฟล์หลักของโปรแกรม
* `Library.txt`
  * รายชื่อแพ็กเกจที่ต้องติดตั้ง
* `Install.bat`
  * ติดตั้ง dependencies และเบราว์เซอร์จำลองสำหรับทะลวงบล็อคเว็บ
* `Run.bat`
  * ใช้รันโปรแกรมบน Windows (แบบซ่อนหน้าต่าง Console)
* `Create_Shortcut.bat`
  * สร้าง Shortcut โปรแกรมพร้อมไอคอนไว้ที่หน้า Desktop
* `icon.ico`
  * ไฟล์รูปภาพไอคอนโปรแกรม

## ความสามารถหลัก

* **โหลดนิยายอัตโนมัติ:** รองรับการดึงเนื้อหาจาก URL หน้าสารบัญ หรือหน้าอ่านนิยาย
* **Bypass Cloudflare:** มีระบบจำลองเบราว์เซอร์เพื่อขอ Cookie ทะลวงหน้าต่างป้องกันบอท
* **จัดการไฟล์เอกสาร:** รวมไฟล์ย่อย, แยกไฟล์ตอน, และแปลงไฟล์ (`.txt`, `.docx`, `.pdf`, `.md`)
* **จัดการคำศัพท์ (Vocab Optimizer):** ตัดคำศัพท์ที่ซ้ำกันออก เลือกระบบการเก็บคำซ้ำได้ (เช่น เก็บอันที่ยาวที่สุด)
* **สร้างวิดีโอนิยายเสียง:** นำไฟล์เสียง (Audio) มาประกอบภาพนิ่งเพื่อเรนเดอร์เป็นไฟล์ `.mp4`
* **หน้าต่าง UI ทันสมัย:** ใช้งานง่าย ไม่ต้องพึ่งพา Command Line รองรับ Dark Mode / Light Mode

## การติดตั้ง

1. ติดตั้ง Python 3 บน Windows (อย่าลืมติ๊ก `Add Python to PATH`)
2. ดับเบิลคลิก `Install.bat`
3. รอให้ระบบติดตั้ง dependencies และเบราว์เซอร์ให้เสร็จสมบูรณ์

หรือใช้คำสั่งผ่าน Command Prompt (กรณีไม่ใช้ไฟล์ .bat):

```bash
pip install customtkinter python-docx PyMuPDF tkinterdnd2 moviepy curl_cffi requests playwright chardet beautifulsoup4
playwright install chromium
```

## การใช้งาน

1. รัน `Run.bat` (หรือดับเบิลคลิก Shortcut ที่ Desktop)
2. เลือกแถบเมนูเครื่องมือที่ต้องการใช้งานจากหน้าต่างหลัก
3. หากเป็นโหมด **ดาวน์โหลดนิยาย** ให้วางลิงก์ เลือกโฟลเดอร์เซฟไฟล์ และตั้งค่ารูปแบบชื่อไฟล์
4. หากเป็นโหมด **จัดการไฟล์/คำศัพท์/วิดีโอ** ให้ลากไฟล์หรือเลือกโฟลเดอร์ที่ต้องการประมวลผล
5. กดปุ่มเริ่มทำงาน และสามารถติดตามสถานะการทำงานได้ที่หน้าต่าง Console Log ด้านล่าง

### โหมดการดาวน์โหลดนิยาย (Novel Downloader)

* **ดึงจากสารบัญ (Concurrent)**
  * ดึงลิงก์ตอนทั้งหมดจากหน้าสารบัญ แล้วโหลดพร้อมกันหลายไฟล์ (รวดเร็ว)
* **ไต่ลิงก์ทีละตอน (Crawler)**
  * อาศัยการกดปุ่ม "หน้าถัดไป" ในเว็บ ชัวร์ 100% สำหรับเว็บที่ดึงสารบัญยาก (เช่น TWKAN)

### ระบบ Bypass Cloudflare คืออะไร

เมื่อเจอเว็บที่มีการป้องกันบอท (เช่น ติดหน้า "Just a moment...") โปรแกรมสามารถใช้ `playwright` เพื่อเปิดเบราว์เซอร์ซ่อนรูปขึ้นมาจัดการได้:

* ใช้รับและเก็บ `Cookie` กับ `User-Agent` ของเว็บนั้นๆ
* นำข้อมูลที่ได้ไปใช้ดาวน์โหลดไฟล์ด้วยความเร็วสูง
* ผู้ใช้สามารถกดปุ่ม "ขอ Cloudflare Cookie" ด้วยตัวเองได้หากโหลดนิยายไม่ผ่าน
